# 印记 - 快速开始指南

## 🚀 5分钟快速启动

### 1. 安装依赖（可选）

```bash
cd backend
pip install -r requirements.txt
```

**注意**: 大部分依赖已安装，只有 `aiosmtplib` 是可选的（会自动降级使用内置smtplib）

### 2. 配置环境变量

配置文件 `.env` 已创建并配置好，默认使用：

```bash
# QQ邮箱（已配置）
QQ_EMAIL=2337590486@qq.com
QQ_EMAIL_AUTH_CODE=lkvxwqnfjwvpdief

# 其他配置使用默认值即可
```

### 3. 启动应用

```bash
cd backend
python main.py
```

**看到以下输出表示启动成功**:
```
INFO:     Uvicorn running on http://0.0.0.0:8000 (Press CTRL+C to quit)
INFO:     Application startup complete.
```

### 4. 访问API文档

打开浏览器访问:
- **Swagger UI**: http://localhost:8000/docs
- **ReDoc**: http://localhost:8000/redoc

## 📝 完整注册流程示例

### 步骤1: 发送注册验证码

**API**: `POST /api/v1/auth/register/send-code`

**请求体**:
```json
{
  "email": "your@email.com",
  "type": "register"
}
```

**响应**:
```json
{
  "success": true,
  "message": "验证码已发送到您的邮箱"
}
```

### 步骤2: 检查邮箱

收件箱会收到主题为"【印记】注册验证码"的邮件，内容包含6位数字验证码，例如：
```
您的注册验证码是：123456
验证码有效期为5分钟，请尽快完成注册。
```

### 步骤3: 注册账号

**API**: `POST /api/v1/auth/register`

**请求体**:
```json
{
  "email": "your@email.com",
  "code": "123456",
  "password": "yourpassword",
  "username": "昵称（可选）"
}
```

**响应**:
```json
{
  "access_token": "eyJ0eXAiOiJKV1QiLCJhbGc...",
  "token_type": "bearer",
  "user": {
    "id": 1,
    "email": "your@email.com",
    "username": "昵称",
    "is_active": true,
    "is_verified": false,
    "created_at": "2026-03-05T14:00:00Z",
    "updated_at": "2026-03-05T14:00:00Z"
  }
}
```

### 步骤4: 保存令牌

复制返回的 `access_token`，后续API调用需要使用。

### 步骤5: 测试认证

**API**: `GET /api/v1/auth/me`

1. 在Swagger UI页面点击右上角 "Authorize" 按钮
2. 输入: `Bearer <你的access_token>`
   - 例如: `Bearer eyJ0eXAiOiJKV1QiLCJhbGc...`
3. 点击 "Authorize"
4. 点击 "Execute" 执行请求

**响应**:
```json
{
  "id": 1,
  "email": "your@email.com",
  "username": "昵称",
  "is_active": true,
  "is_verified": false,
  "created_at": "2026-03-05T14:00:00Z",
  "updated_at": "2026-03-05T14:00:00Z"
}
```

## 🔧 配置调整

### 修改JWT过期时间

编辑 `backend/.env`:
```bash
# 7天（默认）
ACCESS_TOKEN_EXPIRE_MINUTES=10080

# 1天
ACCESS_TOKEN_EXPIRE_MINUTES=1440

# 30天
ACCESS_TOKEN_EXPIRE_MINUTES=43200
```

### 修改验证码过期时间

编辑 `backend/.env`:
```bash
# 5分钟（默认）
VERIFICATION_CODE_EXPIRE_MINUTES=5

# 10分钟
VERIFICATION_CODE_EXPIRE_MINUTES=10
```

### 修改验证码请求频率限制

编辑 `backend/.env`:
```bash
# 5分钟内最多3次（默认）
MAX_CODE_REQUESTS_PER_5MIN=3
```

### 切换到PostgreSQL（部署时）

编辑 `backend/.env`:
```bash
# 注释SQLite
# DATABASE_URL=sqlite+aiosqlite:///./yinji.db

# 启用PostgreSQL
DATABASE_URL=postgresql+asyncpg://user:password@host:5432/yinji
```

## ❓ 常见问题

### Q1: 邮件发送失败怎么办？

**检查**:
1. QQ邮箱是否开启SMTP服务
2. 授权码是否正确（不是QQ密码）
3. 网络是否正常

**获取QQ邮箱授权码**:
1. 登录 https://mail.qq.com
2. 设置 → 账户 → SMTP服务
3. 开启服务并生成授权码

### Q2: 数据库初始化失败？

**解决**:
```bash
# 删除旧数据库
del backend\yinji.db

# 重启应用
python main.py
```

### Q3: 验证码收不到？

**检查**:
1. 垃圾邮件箱
2. 验证码是否过期（5分钟）
3. 是否达到请求频率限制

### Q4: 如何查看详细日志？

编辑 `backend/.env`:
```bash
DEBUG=true
```

重启应用后会看到详细SQL日志。
