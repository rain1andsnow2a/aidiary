# 2026-04-05 开发日志：情绪向量底座升级（本地词典优先 + pysenti可插拔）

## 目标

在不破坏现有 8 维情绪特征向量逻辑的前提下，提升词典覆盖能力与工程可维护性。

## 完成内容

1. `emotion_feature_service.py` 升级为多源词典底座：
   - 支持 `EMOTION_LEXICON_SOURCES=builtin,ntusd,pysenti`
   - 默认本地词典优先（NTUSD），可选融合 `pysenti`
   - 外部词典按权重融合，不做硬覆盖
2. 增强词典读取器：
   - 兼容 `词` / `词+分数`
   - 支持注释行
   - 自动去重
3. 新增 NTUSD 下载脚本：
   - `backend/scripts/fetch_ntusd_lexicons.py`
4. 更新词典说明文档：
   - `backend/app/data/lexicons/README.md`

## 配置示例

```env
EMOTION_EXT_LEXICON_ENABLED=true
EMOTION_LEXICON_SOURCES=builtin,ntusd,pysenti
EMOTION_EXT_LEXICON_WEIGHT=0.65
EMOTION_EXT_LEXICON_POS_PATH=/home/ubuntu/yinji-smart-diary/backend/app/data/lexicons/ntusd_positive_simplified.txt
EMOTION_EXT_LEXICON_NEG_PATH=/home/ubuntu/yinji-smart-diary/backend/app/data/lexicons/ntusd_negative_simplified.txt
```

## 说明

若运行环境未安装 `pysenti`，服务会自动跳过该源，不影响主流程。
