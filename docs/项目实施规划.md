# 印记 - 项目实施规划

## 📊 项目概览

**项目名称**: 印记 (YinJi) - 基于RAG知识库的智能日记应用
**开发阶段**: MVP (最小可行产品)

### Phase 1.2: 邮箱认证功能 ✅

**已实现功能**:
- ✅ 发送注册验证码 (POST /api/v1/auth/register/send-code)
- ✅ 验证注册验证码 (POST /api/v1/auth/register/verify)
- ✅ 用户注册 (POST /api/v1/auth/register)
- ✅ 发送登录验证码 (POST /api/v1/auth/login/send-code)
- ✅ 用户登录 (POST /api/v1/auth/login)
- ✅ 获取当前用户信息 (GET /api/v1/auth/me)
- ✅ 测试邮件发送 (GET /api/v1/auth/test-email)

**技术特性**:
- ✅ QQ SMTP邮件发送（支持SSL）
- ✅ 6位数字验证码
- ✅ 验证码5分钟过期
- ✅ 频率限制（5分钟内最多3次）
- ✅ JWT认证（7天有效期）
- ✅ 密码bcrypt哈希
- ✅ SQLite数据库（可切换PostgreSQL）

**API文档**:
- Swagger UI: http://localhost:8000/docs
- ReDoc: http://localhost:8000/redoc

### Phase 3: AI Agent系统（LangGraph）

**预计时间**: 16-22小时

**Agent架构**:
```
Agent 0: Context Collector
  ├─ 获取用户画像
  ├─ 获取时间轴上下文
  └─ 向量搜索（Milvus，可选）

Agent A: Timeline Manager
  ├─ 提取关键事件
  ├─ 检测连续模式
  └─ 更新timeline_events表

Agent B: Satir Therapist
  ├─ Node B1: 情绪侦探（情绪层）
  ├─ Node B2: 信念挖掘（信念层）
  ├─ Node B3: 灵魂摆渡人（存在层）
  └─ Node B4: 疗愈回信生成器

Agent C: Social Content Creator
  ├─ 生成3版本文案
  └─ 学习用户编辑记录

Agent D: Graph Builder（Phase 4）
  └─ 更新Neo4j知识图谱
```

**依赖服务**:
- DeepSeek V3 API（需要API key）
- Milvus向量数据库（可选，Phase 5）
- Neo4j图数据库（可选，Phase 4）

**API端点**:
```
POST /api/v1/ai/analyze          # 触发分析（异步）
GET  /api/v1/ai/analyze/{taskId} # 获取结果
GET  /api/v1/ai/analyses         # 历史记录
```
