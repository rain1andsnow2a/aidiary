# 社区功能

## 功能概述

用户在社区里可以发布自己的生活日常，支持文字、图片、视频等多种形式。
可以对他人的动态进行点赞、评论、收藏等操作。
每个人都有自己的收藏夹，也有浏览记录（只有点开一个具体的动态才会记录浏览记录，而不是浏览列表时记录）。
用户在社区里可以选择匿名发表想法。

## 匿名机制

匿名帖子不允许再次编辑（保护发布者，也减少恶意修改）。
匿名用户的互动（点赞/评论）也匿名。

## 核心定位

不做开放式广场，做"情绪共鸣圈"。
开放式社区容易变成刷屏、焦虑放大器，心理健康类 app 尤应避免。

建议按"情绪场景"组织社区，而不是时间流：

社区分为6个圈子，用户可以根据自己的情况来在合适的圈子发帖

- [焦虑] 我们都在焦虑的事
- [失落] 那些低落的时刻
- [成长] 突破自我的瞬间
- [平静] 享受当下的时光
- [困惑] 想不明白的时候

不做粉丝数/关心数体系。

匿名发布时强制弹窗提示：“你的内容会被匿名展示，但请记得尊重自己也尊重他人。”

帖子卡片增加“情绪强度”指示器（用户发帖时可选择1-5级），帮助他人判断当前状态。

匿名用户用统一头像+随机色块，但不要用默认灰色头像（太冷漠）。

每个情绪圈子使用不同的柔和色彩（焦虑用淡紫、失落用灰蓝、成长用暖绿、平静用米白、困惑用淡橙），让视觉也传递情绪。

将“点赞”改为“共鸣”，并显示共鸣数。共鸣不需要理由，但可以长按选择共鸣原因，注意是选择原因，而不是填写原因。（“我也是”“抱抱你”“学到了”这些预设选项），这些原因会聚合展示。
