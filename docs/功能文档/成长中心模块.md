# 成长中心模块（Growth Center）- 功能设计文档

> 面向 vibe coding / AI 辅助开发使用。
> 本文档描述「成长中心」页面的完整视觉设计、交互逻辑与数据接入方式。

---

## 一、路由与命名变更

### 当前问题

现有「情绪地形图」功能位于 `/timeline` 路径，名称与路径均不准确：
- 页面标题「成长轨迹」与路由 `/timeline` 不一致
- 功能已不止于「地形图」，而是承载多维度的成长分析

### 变更方案

| 项目 | 旧值 | 新值 |
|------|------|------|
| 路由路径 | `/timeline` | `/growth` |
| 页面标题 | 成长轨迹 / 情绪地形图 | 成长中心 |
| 导航栏菜单文字 | 情绪地形图 | 成长中心 |
| 组件目录 | `pages/Timeline/` | `pages/GrowthCenter/` |

### 迁移注意事项

- 保留 `/timeline` 路由的重定向：`/timeline` → `/growth`（避免旧链接失效）
- 更新所有内部跳转链接（首页快捷入口卡片、日记详情页的「查看成长轨迹」按钮等）

## 二、页面整体结构

```
/growth
├── 页头：标题「成长中心」+ 写日记按钮（右上角）+ 返回按钮（左上角）
├── Section 1：心情日历（Mood Calendar）
├── Section 2：能量值趋势图（Energy Trend）
├── Section 3：萨提亚词云卡片（Satir Keyword Cards）
└── Section 4：人际关系图谱（占位，后续迭代）
```

各 Section 垂直排列，间距 32px，整体背景色与现有页面保持一致（米白 `#FAF9F7`）。

## 三、Section 1：心情日历（Mood Calendar）

### 视觉设计

- 以**月为单位**展示，每天一个圆形色块，7列（周日～周六）排列
- 圆形直径：桌面端 36px，移动端 28px
- 圆形风格：`border-radius: 50%`，不使用硬方块
- 透明度随当天**日记字数或情绪浓度**线性映射：
  - 无日记：`opacity: 0.15`（仅显示淡色占位）
  - 有日记：`opacity: 0.5 ～ 1.0`（字数越多/情绪越强，越不透明）

### 颜色编码（情绪色谱）

| 情绪类型 | 颜色 | Hex 参考 |
|--------|------|---------|
| 平静 / 喜悦 | 琥珀金 | `#F5C842` |
| 成长 / 积极 | 柔绿色 | `#7BC47F` |
| 焦虑 / 压力 | 雾霾蓝 | `#8AAFC8` |
| 困惑 / 低落 | 淡紫色 | `#B8A9D4` |
| 愤怒 / 激动 | 柔橘红 | `#E8856A` |
| 无记录 | 浅灰 | `#E8E6E2` |

> 若当天有多种情绪，取主要情绪（AI 分析时返回的 `primary_emotion` 字段）决定颜色。

### 月份导航

- 左右箭头切换上一月 / 下一月
- 默认展示当前月
- 切换时圆形色块做 **fade + slight scale** 动画过渡（`transition: opacity 0.3s, transform 0.2s`）

### 交互

**悬停（Hover）**：
- 触发浮窗 Tooltip，内容由后端 AI 生成，格式：
  ```
  "{日期}" · {主要情绪}
  "{映记精灵的一句简短总结，≤20字}"
  ```
  示例：*"3月22日 · 平静 / 这天你去爬山了，感觉很舒畅。"*
- Tooltip 数据在页面加载时批量预拉取（当月所有有日记的日期），缓存在前端状态，避免 hover 时实时请求

**点击**：
- 跳转至该日日记详情页

## 四、Section 2：能量值趋势图（Energy Trend）

> 在现有情绪地形图的折线图基础上进行视觉升级，**不替换，而是重构样式**。

### 视觉升级要点

- 图表类型：由折线图改为**平滑曲线面积图（Spline Area Chart）**
  - 使用 Recharts 的 `AreaChart` + `type="monotone"`
  - 曲线本身加粗至 2.5px，加圆角端点
- 渐变填充（`linearGradient`）：
  - 能量值高（≥7）区域：琥珀金渐变 `#F5C842` → 透明
  - 能量值中（4～6）区域：柔粉渐变（与主页 Banner 一致）
  - 能量值低（≤3）区域：藏青蓝渐变 `#4A6FA5` → 透明
  - 实现方式：根据数值区间动态切换 `fill` 的 `gradientId`

### 时间范围切换

- 保留现有 7天 / 30天 / 90天 Tab
- 切换时曲线做**平滑过渡动画**（Recharts 内置 `animationDuration: 600`）

### 关键事件标注（RAG 驱动）

仅对**单日能量变化幅度 ≥ 2 的波峰/波谷节点**进行标注，避免视觉过载。

- 标注形式：曲线节点处渲染一个小图标（emoji 或 SVG icon）
- 图标由 AI 根据当天日记关键事件自动选取，示例：
  - 爬山 → ⛰️
  - 加班 → 💼
  - 失眠 → 🌙
  - 聚会 → 🎉
- 点击标注图标：弹出小卡片，显示事件摘要 + 「查看日记」链接

### 标注数据接口示例

```
GET /api/growth/energy?range=30
Response:
{
  "series": [
    { "date": "2026-03-22", "energy": 7.0, "event_icon": "⛰️", "event_summary": "去爬山，状态很好" },
    { "date": "2026-03-25", "energy": 4.5, "event_icon": null, "event_summary": null },
    ...
  ]
}
```

> `event_icon` 和 `event_summary` 仅在变化幅度达到阈值时由后端 AI 填充，其余为 `null`。

## 五、Section 3：萨提亚词云卡片（Satir Keyword Cards）

> **设计决策说明**：不采用传统词云（视觉杂乱、难以传达深层含义），改为「萨提亚冰山关键词卡片」，按冰山层级分区展示高频且重要的关键词。

### 布局

水平排列 5 张卡片（对应萨提亚冰山 5 层），移动端改为竖向滚动：

```
[ 行为层 ] [ 情绪层 ] [ 认知层 ] [ 信念层 ] [ 渴望层 ]
```

每张卡片高度约 160px，宽度等分，圆角 16px，卡片背景为该层级对应的淡色。

### 各层级视觉规范

| 层级 | 标题 | 背景色 | 关键词颜色 |
|-----|------|-------|----------|
| 行为层 | 我做了什么 | `#FFF5E6` | `#D4820A` |
| 情绪层 | 我感受到 | `#F0EBF8` | `#7B5EA7` |
| 认知层 | 我在想 | `#E8F4F0` | `#2E7D62` |
| 信念层 | 我相信 | `#EBF2FA` | `#2563AB` |
| 渴望层 | 我渴望 | `#FEF3F2` | `#C0392B` |

### 关键词展示规则

- 每张卡片展示 Top 5～8 个关键词
- **字体大小** = 词频权重 × 重要性评分（RAG 返回），范围 12px ～ 22px
- **字体颜色深浅** = 情绪正负向（正向偏亮，负向偏深）
- 词语之间使用自然流动排列（`flexWrap: wrap`），不强制对齐

### 分析时间范围

- 默认分析最近 **30 天**日记
- 提供 30天 / 90天 切换 Tab，与趋势图联动（切换时两个组件同步刷新）

### 数据接口示例

```
GET /api/growth/satir-keywords?range=30
Response:
{
  "layers": {
    "behavior": [{ "word": "加班", "weight": 0.85, "sentiment": -0.3 }, ...],
    "emotion":  [{ "word": "焦虑", "weight": 0.90, "sentiment": -0.7 }, ...],
    "cognition":[{ "word": "我不够好", "weight": 0.70, "sentiment": -0.6 }, ...],
    "belief":   [{ "word": "努力才有价值", "weight": 0.65, "sentiment": 0.1 }, ...],
    "desire":   [{ "word": "被看见", "weight": 0.80, "sentiment": 0.5 }, ...]
  }
}
```

> 后端通过 RAG 检索用户日记 + AI分类，将关键词归入对应萨提亚层级。

## 九、整体交互联动

- 「心情日历」点击某天 → 跳转日记详情
- 「能量趋势图」点击事件标注 → 跳转日记详情
- 「萨提亚词云卡片」时间范围切换 → 能量趋势图同步切换时间范围
- 映记精灵（AI 助手）在此页面可感知当前展示数据，支持用户提问："我最近渴望什么？" 等

---

## 十、边界情况处理

| 情况 | 处理方式 |
|-----|---------|
| 日记数量 < 3 条 | 展示引导提示："再写几篇日记，成长洞察将逐渐清晰" |
| 当月无日记 | 心情日历全部显示为淡灰占位圆，不报错 |
| 能量值无突变节点 | 趋势图不显示任何事件标注，仅展示平滑曲线 |
