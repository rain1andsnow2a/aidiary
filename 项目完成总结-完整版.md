## 🛠️ 技术栈

### 后端
- **框架**: FastAPI 0.104.1
- **数据库**: SQLite 3 / PostgreSQL 16
- **ORM**: SQLAlchemy 2.0
- **认证**: JWT + bcrypt
- **邮件**: QQ SMTP
- **AI**: DeepSeek API
- **Agent系统**: 自定义多Agent架构

### 前端
- **框架**: React 18.3.1
- **语言**: TypeScript 5.4.5
- **构建**: Vite 5.2.8
- **UI**: Tailwind CSS + shadcn/ui
- **状态**: Zustand 4.5.2
- **路由**: React Router 6.22.3
- **HTTP**: Axios 1.6.8
- **图表**: Recharts 2.12.5
- **日期**: date-fns 3.6.0

## 🚀 部署指南

### 后端部署

```bash
cd backend
pip install -r requirements.txt
python main.py
```

访问: http://localhost:8000

### 前端部署

```bash
cd frontend
npm install
npm run dev
```
